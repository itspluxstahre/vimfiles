Installation
============

Grab the plugin submodules:
`cd ~/.vim && git submodule init && git submodule update`


Make sure vim finds the vimrc file by either symlinking it:
`ln -s ~/.vim/vimrc ~/.vimrc`

or by sourcing it from  your own ~/.vimrc:
`source ~/.vim/vimrc`
